# Progress Log

## Learnings
(Patterns discovered during implementation)

- pytest is not installed in the venv; must install with `python -m pip install pytest` first
- The autonomy tests fail due to missing `aiosqlite`; ignore those test files when running broad test passes
- Secrets always come from `os.getenv()` in `build_app_config`; non-secrets come from `_get_nested(json_config, ...)`
- `DEFAULT_CONFIG` in `config_manager.py` and the mapping in `build_app_config` in `config.py` must both be updated for new config sections
